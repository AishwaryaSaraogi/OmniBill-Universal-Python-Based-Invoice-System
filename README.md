# OmniBill Universal Invoice System

A powerful and elegant Python-based Invoice Generation System built with Streamlit and FPDF. It allows you to manage student registrations, dynamically calculate course fees, and generate clean PDF invoices on the fly.

---

## 🚀 How to Run the Project Locally

Follow these simple steps to set up and run the application on your computer:

### 🔹 Step 1: Clone or Download the Project
Download all the core project files into a single, dedicated folder on your local machine:
* `Bill_Generator.py`
* `Students.csv`
* `Courses.csv`
* `Logo.png`
* `Requirements.txt`

### 🔹 Step 2: Install Required Libraries
Open your terminal or command prompt inside that specific project folder and execute the following command to install all necessary dependencies:
`pip install -r Requirements.txt`

### 🔹 Step 3: Start the Application
Now, run the Streamlit server to launch the app in your browser:
* `cd "path/to/your/project-folder"`
* `streamlit run Bill_Generator.py`

---

🛠️ Tech Stack Used
* Frontend/UI: Streamlit
* Database/Data Handling: Pandas (CSV)
* PDF Generation: FPDF
